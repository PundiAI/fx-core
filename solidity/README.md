## Overview

### Installation

```
$ yarn add @functionx_io/contracts
```

### Testing

```
# Run normal tests
$ yarn test

# Run tests with fork network

# 1. Start a local node fork network

$ yarn fork:xxxx

# 2. Run tests with fork network

$ yarn fork:test

```
